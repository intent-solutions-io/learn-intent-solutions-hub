The toolkit

Claude Code, Cowork, and plugins. Install once, then get back to the work.

01 — Claude Code

Anthropic's agentic CLI

Runs in the terminal. Also available as a desktop app, on the web at claude.ai/code, and as IDE extensions for VS Code and JetBrains.

Install

$ npm install -g @anthropic-ai/claude-code

Then run claude in a project directory to start. It authenticates on first run.

By default it asks permission before actions that change files or run commands. That is expected. Permission modes are configurable in settings.

Full reference: docs.claude.com

02 — Claude Cowork

The collaborative surface

A session-based collaborative environment used alongside Claude Code. Details differ from the CLI flow above.
