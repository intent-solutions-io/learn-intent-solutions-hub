Tools & setup

Claude Code, Cowork, and plugins. Install once, then get back to the work.

01 — Claude Code

Anthropic's agentic CLI

Runs in the terminal. Also available as a desktop app, on the web at claude.ai/code, and as IDE extensions for VS Code and JetBrains.

Install
$ npm install -g @anthropic-ai/claude-code

Then run claude in a project directory to start. It authenticates on first run. By default it asks permission before actions that change files or run commands — that's expected, and configurable in settings.

Full reference: docs.claude.com

02 — Claude Cowork

The collaborative surface

A session-based collaborative environment used alongside Claude Code. Details differ from the CLI flow above.

Setup and usage: docs.claude.com

Extending Claude Code

Good to know
Is certification guaranteed if I finish the practice tests?
No. Intent Solutions provides preparation guidance — the exam itself is administered and scored by Anthropic. Practice-test readiness is a study signal, not an eligibility guarantee.
Official vs. Intent Solutions guidance — what's the difference?
"Official verified" is sourced and cited against Anthropic's own documentation. "Intent Solutions recommendation" is our judgment and community experience — useful, but never a substitute for the former.
Who do I ask about partner pricing or eligibility?
Ask on the right — partner-policy questions route to Jeremy in one batch rather than everyone contacting Anthropic support individually.
